# 输出成果目录说明

本目录包含中期阶段所有分析成果。每份成果的方法、推理、结果详细说明如下。

---

## figures/ — 可视化成果

### 1. culture_genealogy_tree.html — 文化谱系树可视化

- **展示内容**：南海区8大文化门类→24子类→97条目的层级结构
- **研究方法**：综合非遗名录（权威分类）+ 文化典籍（历史纵深）+ NER实体频次（数据验证），以三者互校确定分类
- **技术实现**：ECharts Tree组件，LR（左到右）布局，初始展开2层，暗色主题
- **交互功能**：点击节点展开/折叠子树，鼠标悬停显示描述信息
- **核心结论**：南海区文化以武术文化（醒狮/咏春/龙舟）和建筑文化（古村/祠堂/书院）为两大支柱，工艺文化（藤编/金箔/缫丝等）类型最丰富（5个子类18个条目）

### 2. scenic_genealogy_tree.html — 景点谱系树可视化

- **展示内容**：1,353个景点按 类型→体验度(高/中/低)→具体景点 的三层结构
- **研究方法**：五维加权评分模型——平台评分(30%) + 好评率(25%) + 文化深度(20%) + 评论活跃度(15%) + 照片丰富度(10%)
- **核心结论**：高级=0、中级=483（35.7%）、低级=870（64.3%）。无景点达到"高"级，说明南海区文旅产品普遍缺少"高评分+丰富评论+强文化关联"的综合优质体验

### 3. scenic_town_tree.html — 景点镇街分布可视化

- **展示内容**：1,353个景点按 镇街→类别→具体景点 的分布
- **核心结论**：桂城街道(327)独占鳌头，九江镇(47)最少但非遗最密集

### 4. knowledge_graph.html — 文旅知识图谱可视化

- **展示内容**：150个节点（11种类型）、224条关系的力导向交互式图谱
- **研究方法**：多源数据融合——NER文化实体(TOP80) + POI景点(TOP30) + 非遗(26) + 镇街(7) + 文化门类(8)，通过5类关系（共现/位于/承载/传承/包含）交叉关联
- **推理价值**：图谱直观揭示了"文化要素↔非遗项目↔传承基地↔所在镇街"的关联链条。孤立节点（无边连接）代表"尚未融入文旅体系"的文化要素，是潜在的开发对象
- **交互功能**：拖拽节点、缩放、鼠标悬停显示类型信息、相邻节点高亮

### 5. coupling_analysis.html — 双谱系耦合分析可视化

- **展示内容**：左侧为文化-景点关系力导向图，右侧为统计面板（数字卡片+核心发现+优化建议+缺失列表）
- **研究方法**：四种实体对齐类型（实体对应/文化承载/空间共存/主题关联）→ 四种耦合状态判定（强耦合/错位/缺失A/缺失B）
- **核心结论**：
  - 绿色实线=强耦合16对（融合成功）
  - 橙色虚线=错位9对（有关联但深度不足）
  - 红色孤立节点=缺失A14项（文化未转化为旅游产品）
  - 蓝色孤立节点=缺失B6个（景点缺少文化灵魂）
- **五大建议**：开发省级非遗旅游化/现代公园注入非遗/书院文化深度融合/九江美食集群/古村连线路线

### 6. spatial_analysis.html — 空间分析四合一可视化

- **四个子图**：
  - 左上-散点图：1,353个POI(青色)+26个非遗(红色)的经纬度分布
  - 右上-柱状图：7个镇街的POI和非遗资源数量堆叠对比
  - 左下-热力图：POI核密度分布（颜色从深蓝→紫→橙→黄 = 低→高密度）
  - 右下-雷达图：TOP5镇街的多维度资源对比
- **研究方法**：Epanechnikov核密度估计(带宽3km) + DBSCAN聚类(eps=3km) + 镇街统计
- **核心结论**：资源空间呈"中心城区-西南文化带"二元格局，九江镇为典型"文化富集-旅游洼地"

## tables/ — 统计表格
