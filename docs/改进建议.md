# 中期阶段改进建议 & 解决方案

## 问题1：文化实体与空间无法落位

**原始问题**：具体的文化点无法对应到具体的POI数据，无法实现文化-空间的精准对应。

**已实施的解决方案**：

1. **引入政府普查级文化载体锚点数据**（`辅助补充数据/wenhuadian/`）
   - 不可移动文物 shapefile：80处（含坐标、年代、保护级别）
   - 非遗项目 shapefile：36项（含类型、级别、乡镇）
   - 文化景观 shapefile：19处
   - 名镇名村传统村落 shapefile：12处
   - 圩市街区 shapefile：18处
   - **合计165条文化载体锚点**，全部有精确坐标

2. **构建 `cultural_anchors.json` 锚点表**
   - 每个锚点包含：名称、类型、年代、保护级别、镇街归属、经纬度
   - 自动注入NER自定义词典，确保典籍文本中能识别这些锚点名称

3. **POI空间关联匹配**（500m半径）
   - 对13,113条POI逐一计算与165个文化锚点的距离
   - **996条POI**已成功关联到至少一个文化载体锚点
   - 实现了"文化点 → 具体POI"的精准对应

**效果**：文化实体空间落位率从 0% → 7.6%（996/13113），核心文化区域（西樵/九江/丹灶）的落位率更高。

## 问题2：评分评论数据不足

**原始问题**：评论数据太少，评分数据没有足够的样本量，缺乏说服力。

**已实施的解决方案**：

1. **融合三平台评论数据**（`辅助补充数据/携程去哪儿马蜂窝评价/`）
   - 携程：14,161条
   - 去哪儿：723条
   - 马蜂窝：559条
   - **合计15,443条**新增评论

2. **融合2024佛山市POI shapefile**
   - 南海区273,518条POI中筛选12,906条文旅相关POI
   - POI总量从1,353 → 13,113（近10倍提升）

3. **评论覆盖景点数**
   - 原始：867个景点有评论
   - 融合后：**3,879个景点有评论**（4.5倍提升）
   - 体验度评估：高=59 / 中=561 / 低=12,493

**效果**：评论覆盖率从6.4%提升到29.6%，核心景点评论条数显著增加。

## 问题3：知识图谱混乱，缺乏语义关系

**原始问题**：知识图谱节点含大量非文化内容，无法描述实体间的具体关系。

**已实施的解决方案**：

1. **NER严格文化过滤**
   - 彻底取消"其他"类实体，只保留6类文化实体
   - 626个纯文化实体（人物95 / 地点206 / 建筑116 / 朝代114 / 文化要素88 / 事件7）

2. **三层文化载体锚定结构**
   - **上层（典籍文化层）**：NER提取的文化实体 → 100个
   - **中间层（文化载体锚点层）**：165条官方文化载体 + 非遗53项 → 182个
   - **下层（旅游产品层）**：POI景点 + 镇街 → 105个
   - **合计387个节点**，全部文化相关

3. **10种语义关系**
   - 共现关联(1576) / 对应景点(242) / 位于(159) / 同时代(83) / 传承于(26) 等
   - **合计2,118条关系**，每条都有明确的关系类型标注

4. **交互式信息卡**
   - 悬停节点显示：名称 + 类型标签 + 层级标签 + 详细介绍
   - 悬停连线显示：两端节点 + 关系类型

**效果**：图谱从"杂乱的力导向图"变为"三层文化载体锚定结构"，节点全部与南海文旅相关。
